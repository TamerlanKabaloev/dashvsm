# Онлайн Шахматы

Веб-приложение для игры в шахматы с другом по ссылке в стиле lichess.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Приложение будет доступно по адресу: `http://localhost:5000`

## Использование

1. Откройте приложение в браузере
2. Нажмите "Создать игру" - будет создана новая игра и сгенерирована ссылка
3. Поделитесь ссылкой с другом
4. Когда друг откроет ссылку, игра начнется автоматически

## Особенности

- Реальное время через WebSocket
- Красивый интерфейс в стиле lichess
- Поддержка всех правил шахмат
- История ходов
- Индикация возможных ходов
- Подсветка последнего хода
- Индикация шаха

## Технологии

- Flask (Python веб-фреймворк)
- Flask-SocketIO (WebSocket поддержка)
- Chess.js (JavaScript библиотека для логики шахмат)
- HTML/CSS/JavaScript
